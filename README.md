# 🖼️ Image Gallery Web App

A responsive and interactive Image Gallery built using **HTML, CSS, and Vanilla JavaScript**.  
This project was developed as part of the **CodeAlpha Internship Task**.

---

## 🚀 Features

- 📸 Dynamic image gallery rendering from JavaScript data
- 🔍 Category filtering (Travel, Technology, Lifestyle)
- 🖼️ Lightbox view for images
- ⬅️➡️ Next / Previous navigation
- ⌨️ Keyboard support (Arrow keys + ESC)
- 📱 Fully responsive design
- ⚡ Smooth hover & transition effects

---

## 🛠️ Technologies Used

- HTML5
- CSS3 (Grid + Flexbox + Responsive Design)
- JavaScript (ES6)
- Unsplash API images (static links)

---

## 🎯 How It Works

- Images are stored in `data.js`
- `main.js` dynamically renders the gallery
- Filter buttons update displayed images by category
- Clicking an image opens a lightbox
- Users can navigate using buttons or keyboard
